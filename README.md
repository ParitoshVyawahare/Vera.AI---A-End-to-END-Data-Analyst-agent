# Vera.AI

**Your AI data analyst. Fluent in pandas, allergic to guesses.**

Upload a CSV, Excel, Parquet, or JSON file. Vera automatically profiles your data, computes statistics, detects outliers, and writes a full analytical report — grounded strictly in the numbers she sees. Then ask her follow-up questions in plain English and she'll write and safely execute pandas code to answer.

## Features

- Auto-profiling — types, missingness, cardinality
- Statistical analysis — summary stats, correlations, IQR + Isolation Forest outlier detection
- AI-written reports — structured Markdown briefings generated by Groq (Llama 3.3 70B)
- Auto-generated charts — heatmaps, scatters, histograms, bar charts
- Q&A chat with sandboxed code execution — Vera writes pandas code, runs it safely, and returns real answers
- Safety-first — AST-based code sandbox rejects imports, file I/O, and dunder escapes

## Tech Stack

Streamlit · Pandas · scikit-learn · Plotly · LangChain · Groq (Llama 3.3 70B)
